# Task 02: Environment Variables Setup

## Overview

Set up the required environment variables in .env file for the bot to function.

## Core Logic

- Create .env file with required variables
- BOT_TOKEN for Telegram bot authentication
- BOT_ADMIN_TELEGRAM_ID for super admin identification
- PROXY_URL (optional) for Cloudflare workers proxy

## Relations to Code Files

- .env
- .env.example (if needed)

## Steps

1. Create .env file
2. Add BOT_TOKEN variable
3. Add BOT_ADMIN_TELEGRAM_ID variable
4. Add PROXY_URL (optional)

## Checklist

- [ ] .env file created
- [ ] BOT_TOKEN configured
- [ ] BOT_ADMIN_TELEGRAM_ID configured
- [ ] PROXY_URL configured (if needed)
